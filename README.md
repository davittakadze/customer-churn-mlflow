# Customer Churn Prediction System

ეს არის მანქანური სწავლების საბაზისო პროექტი, რომელიც მიზნად ისახავს მომხმარებელთა გადინების (**Customer Churn**) პროგნოზირებას.
პროექტი აგებულია **EDA-ს (Exploratory Data Analysis)** შედეგებზე დაყრდნობით, იყენებს **MLflow**-ს ექსპერიმენტებისა და მოდელების მენეჯმენტისთვის, ხოლო მოდელის სერვინგისთვის (Model Serving) გამოყენებულია **FastAPI**.

## 🛠️ გამოყენებული ტექნოლოგიები
* **Python** (ვერსია 3.10+)
* **FastAPI** & **Uvicorn** (API სერვისისთვის)
* **MLflow** (მოდელების თრექინგისთვის)
* **Scikit-learn** (ლოგისტიკური რეგრესიის მოდელისთვის)
* **Pandas** & **Pydantic** (მონაცემთა ვალიდაციისა და მანიპულაციისთვის)
